    AccountingDocuments

    Arquivo SQL de origem - Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - (ecc) - AccountingDocuments

    Visão geral

    Esta visão é uma visão de relatórios para sistemas SAP que inverte transações de "débito/crédito" com "positivo" para débito e "negativo" para crédito e fornece uma visão abrangente dos dados contábeis.

    Fontes de Dados

    Tabela/Visão Descrição
    BSEG Itens da Linha do Documento Contábil
    BKPF Cabeçalho do Documento Contábil
    TCURXWAERS Decimal da Moeda para Moeda do Documento
    TCURXHWAER Decimal da Moeda para Moeda Local
    TCURXHWAE2 Decimal da Moeda para Segunda Moeda Local
    TCURXHWAE3 Decimal da Moeda para Terceira Moeda Local
    TCURXPSWSL Decimal da Moeda para Números da Transação do Razão Geral
    CalendarDateDimension_BUDAT Dimensão da Data do Calendário para Data de Lançamento
    CalendarDateDimension_BLDAT Dimensão da Data do Calendário para Data do Documento

    Casos de Uso

    • Pode ser usado para analisar dados contábeis, como itens em aberto, faturas vencidas e saldos de clientes.
    • Pode ser usado para criar relatórios sobre desempenho financeiro, como declarações de lucros e perdas e balanços patrimoniais.
    • Pode ser usado para detalhar as transações contábeis individuais.

    Lógica da Query

    A visão usa uma subconsulta para inverter transações de "débito/crédito" com "positivo" para débito e "negativo" para crédito. Em seguida, ele une a subconsulta com a tabela BKPF nos campos MANDT, BUKRS, GJAHR e BELNR. A visão também se junta a várias tabelas decimais de moeda para corrigir a casa decimal de valores para moedas não decimais. Finalmente, a visão se junta à tabela calendar_date_dim para adicionar informações de dimensão de data.

    Possível relação com transações SAP

    Código da Transação Descrição da Transação
    FB03 Exibir Documento
    FB05 Exibir Item de Linha
    FBL1N Exibir Itens de Linha: Contas do Razão Geral
    FBL3N Exibir Itens de Linha: Cliente
    FBL5N Exibir Itens de Linha: Fornecedor
