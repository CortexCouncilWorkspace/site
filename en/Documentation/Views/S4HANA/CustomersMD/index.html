    Google Cloud Cortex Foudantion - Cortex Reporting - (s4) - CustomersMD

    Overview

    This view provides a comprehensive reporting dataset for SAP systems, specifically tailored for customer-related information. It combines data from the KNA1 and ADRC tables to offer a holistic view of customer details, addresses, and other relevant attributes.

    Data Sources

    Source Table/View Description
    KNA1 Customer Master Table
    ADRC Address Table

    Key Use Cases

    • Customer Relationship Management (CRM)
    • Sales and Distribution (SD)
    • Financial Accounting (FI)
    • Controlling (CO)
    • Materials Management (MM)

    Query Logic

    The view joins the KNA1 and ADRC tables using the ADRNR field, ensuring that customer data is linked to their respective addresses. The join condition also filters out any inactive or outdated addresses by selecting records where the DATE_TO field in ADRC is set to '9999-12-31'. The resulting dataset provides a consolidated view of current and relevant customer information.

    Possible relation with SAP Transactions

    Transaction Code Transaction Description
    XD01 Create Customer
    XD02 Change Customer
    XD03 Display Customer
    FBL5N Customer Line Items
    VA01 Create Sales Order
