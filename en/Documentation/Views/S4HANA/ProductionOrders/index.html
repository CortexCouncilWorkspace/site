    ProductionOrders

    Source SQL File - Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - (s4) - ProductionOrders

    Overview

    This view provides a comprehensive overview of SAP production orders and their associated items. It combines data from the AFKO, AFPO, AUFK, JEST, and TJ02T tables to provide a holistic view of order-related information, including order status, scheduling details, material requirements, and cost information.

    Data Sources

    Source Table/View Description
    AFKO Production Order Header
    AFPO Production Order Item
    AUFK Order Master Data
    JEST Object Status
    TJ02T Object Status Texts

    Key Use Cases

    • Monitor production order status and progress
    • Analyze order scheduling and capacity planning
    • Track material requirements and availability
    • Review cost estimates and actual costs
    • Identify bottlenecks and areas for improvement

    Query Logic

    The view combines data from multiple tables using INNER JOINs to create a comprehensive dataset. The query logic ensures that only relevant data is included and that the relationships between the tables are maintained.
