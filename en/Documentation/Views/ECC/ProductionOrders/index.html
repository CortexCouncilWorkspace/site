    ProductionOrders

    Source SQL File - Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - (ecc) - ProductionOrders

    Overview

    This view is a reporting view for SAP systems that provides detailed information about production orders and their associated items. It combines data from multiple SAP tables, including AFKO, AFPO, AUFK, JEST, and TJ02T, to provide a comprehensive view of order-related data.

    Data Sources

    Source Table/View Description
    AFKO Header data for production orders, including general information, scheduling data, and status information.
    AFPO Item data for production orders, including information about individual order items, such as material, quantity, and delivery dates.
    AUFK Header data for orders, including general information, cost information, and status information.
    JEST Object status data, including information about the status of an object and its change history.
    TJ02T Text table for object status, providing short and long descriptions of object statuses.

    Key Use Cases

    • This view can be used to analyze production order data, such as order status, scheduling, and cost information.
    • It can also be used to track the progress of production orders and identify any potential issues or delays.
    • Additionally, this view can be used to generate reports on production order data for various purposes, such as production planning, cost analysis, and quality control.

    Query Logic

    The query combines data from the AFKO, AFPO, AUFK, JEST, and TJ02T tables using inner joins. The join conditions ensure that the data from each table is properly linked and that the resulting view contains only relevant data for production orders.
