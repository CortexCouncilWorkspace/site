    SalesOrders

    Archivo SQL de origen: Google Cloud Cortex Data Foundation

    Google Cloud Cortex Foudantion - Cortex Reporting - (s4) - SalesOrders

    Descripción general

    Esta vista proporciona un conjunto de datos completo para informes de órdenes de venta de SAP. Combina datos de múltiples tablas de SAP para proporcionar una vista holística de las órdenes de venta, incluyendo información a nivel de encabezado y de artículo, precios, detalles del cliente y más. La vista está diseñada para soportar varias necesidades de informes y análisis de negocios.

    Fuentes de datos

    Tabla/vista de origen Descripción
    {{ project_id_src }}.{{ dataset_cdc_processed_s4 }}.vbak Tabla de Cabecera de Orden de Venta
    {{ project_id_src }}.{{ dataset_cdc_processed_s4 }}.vbap Tabla de Artículos de Orden de Venta
    {{ project_id_src }}.{{ dataset_cdc_processed_s4 }}.prcd_elements Tabla de Elementos de Precio
    {{ project_id_src }}.{{ dataset_cdc_processed_s4 }}.vbep Tabla de Cantidades Confirmadas de Pedido
    {{ project_id_src }}.{{ dataset_cdc_processed_s4 }}.kna1 Tabla de Datos Maestros de Cliente
    TCURX Tabla de Tipos de Cambio de Moneda

    Casos de uso clave

    • Análisis e informes de órdenes de venta
    • Gestión de relaciones con clientes (CRM)
    • Gestión de inventario
    • Informes financieros
    • Previsión de ventas

    Lógica de consulta

    La vista combina datos de las tablas VBAK, VBAP, PRCD_ELEMENTS, VBEP y KNA1 utilizando INNER JOINs y LEFT OUTER JOINs. También incluye cálculos para el valor neto, el costo y el importe del impuesto en la moneda del documento utilizando la tabla TCURX para la conversión de moneda.

    Posible relación con SAP Transactions

    Codigo de transacción Descripción de la transacción

    VA02 Cambiar Orden de Venta
    VA03 Mostrar Orden de Venta
    VL01N Crear Entrega
    VF01 Crear Factura
